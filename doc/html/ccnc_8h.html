Function documentation

void initMotor(struct MotorPins* motorPins, struct MotorGPIOD* motorGPIOD, struct gpiod_chip* chip)

Initializes a A4988-style motor driver.

Parameters
motorPins Motor driver GPIO pins.
motorGPIOD Uninitialized pointer to a MotorGPIOD instance that is to be populated.
chip Pointer to the GPIOD chip.

Calls GPIOD and request pin output access as the "cnccontroller" consumer. Initial pin values are set to low.

void releaseMotor(struct MotorGPIOD* motorGPIOD)

Safety releases all GPIOD connections for a given motor.

Parameters
motorGPIOD Pointer to the MotorGPIOD instance to be released.

void prepareSubcommand(uint scmd, struct MotorGPIOD* motorGPIOD)

Prepares pin states for a subcommand.

Parameters
scmd A 5-bit subcommand.
motorGPIOD Pointer to the MotorGPIOD instance.

A subcommand is a 5-bit value with the following build-up:

Address (LE)AbbreviationMeaning
0MS3Microstep 3
1MS2Microstep 2
2MS1Microstep 1
3DIRDirection
4ENAEnable

void executeSubcommand(uint scmd, struct MotorGPIOD* motorGPIOD, uint wait)

Executes a subcommand with a specified wait time between step pin high and low.

Parameters
scmd A 5-bit subcommand.
motorGPIOD Pointer to the MotorGPIOD instance.
wait Wait time in microseconds.

prepareSubcommand

void executeSubcommandsOnMotors(uint scmds[], struct MotorGPIOD* motorGPIODs[], uint wait, uint number)

Executes multiple subcommands on multiple motors (one per motor).

Parameters
scmds An array of 5-bit subcommands.
motorGPIODs An array of pointers to MotorGPIOD instances.
wait Wait time in microseconds.
number Number of motors to be commanded.

Computes integer parity.

Parameters

uint addParityBit(uint i)

Prepends parity bit to an integer.

Parameters
i Integer.
Returns Integer with prepended parity bit.

Parity bit is added at address 0 (LE).

Checks if parity bit is properly set.
